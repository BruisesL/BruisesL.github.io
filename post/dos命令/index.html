DOS命令

Dos命令

#盘符切换 
C:
#查看当前目录下的所有文件 
dir
#切换目录 cd (change directory)实例目录为F盘下user文件夹
跨盘符切换 cd /d f:\user 
返回上一级 cd ..
进入目录 cd user 
#清理屏幕 
cls (clear screen)
#退出终端 
exit
#查看电脑ip 
ipconfig
#ping 命令（获取网站ip）
ping www.baidu.com

#打开电脑工具（直接输入软件名称缩写）
clac 计算器
mspaint 画图
notepad 记事本

#文件操作
md test 创建文件夹（文件夹名称tese）
cd>a.txt 创建文件（文件为名字是a的txt文件）
del a.txt 删除文件
rd test 删除文件夹（需进入有该文件夹的目录再进行删除）
